# Habits & Todo Tracker (Mobile App)

Кроссплатформенное мобильное приложение для управления продуктивностью: совмещает в себе трекер привычек и список ежедневных задач.

## Стек технологий
* **Core:** Vue 3 (Composition API)
* **UI:** Ionic Framework
* **Engine:** Capacitor (Android / iOS)
* **Build:** Vite
* **Persistence:** LocalStorage

## Ключевые фичи
* **Interactive Calendar History:** Интерактивный календарь для каждой привычки, позволяющий отслеживать историю выполнений по датам.
* **Todo & Habits Integration:** Разделение на разовые задачи и повторяющиеся привычки.
* **Data Persistence:** Полное сохранение прогресса в LocalStorage.
* **Ionic UI:** Использование нативных компонентов Ionic для быстрого и плавного интерфейса.
